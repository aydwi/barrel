A variadic constructor for BrewCommand.

The singular (almighty) constructor offered by BrewCommand. Due to it being variadic, an arbitrary number of arguments can be passed to the constructor. Barrel expects you to pass a chain of Homebrew arguments that follow your choice of command thereby allowing any possible Homebrew command to be executed.

Parameters
brewAn object of type Brew
cmdThe brew command you want to execute
argsA parameter pack representing in order, the chain of options/text that follow cmd as prescribed by Homebrew. The following object types are allowed in the parameter pack: const char*, std::string
