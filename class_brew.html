Brew::Brew ( )

Default constructor for Brew. Roll the spartan way.

The simplest way to construct a Brew object. Barrel defaults the target architecture to BrewTargetArch::X86_64 and correspondingly the Homebrew installation path to BrewSpec::_BREW_DEFAULT_PATH_X86_64.
